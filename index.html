¿Qué es el Transformer?

Es una arquitectura de red neuronal que prescinde totalmente de la **recurrencia (RNN)** y las **convoluciones (CNN)**. Se basa exclusivamente en mecanismos de atención para establecer dependencias globales entre la entrada y la salida.

Paralelización: A diferencia de las RNN, puede procesar secuencias completas a la vez.

Eficiencia: Requiere significativamente menos tiempo de entrenamiento.

Estructura simplificada Encoder-Decoder

Mecanismo de Auto-Atención (Self-Attention)

Scaled Dot-Product

Calcula la relevancia de cada palabra respecto a las demás usando tres vectores: Queries (Q), Keys (K) y Values (V).

Attention(Q,K,V) = softmax(QKᵀ / √dₖ)V

Multi-Head Attention

Permite al modelo atender simultáneamente a información de diferentes subespacios de representación en distintas posiciones.

Positional Encoding

Como no hay recurrencia, se inyecta información sobre el orden relativo de las palabras usando funciones seno y coseno.

Arquitectura del Modelo

1

Encoder (Izquierda)

Compuesto por un stack de 6 capas idénticas. Cada capa tiene dos subcapas: Multi-head attention y una red Feed-Forward simple por posición.

2

Decoder (Derecha)

También de 6 capas. Incluye una tercera subcapa que realiza atención sobre la salida del Encoder para guiar la generación.

3

Conexiones Residuales

Cada subcapa emplea una conexión residual seguida de Normalización de Capa (Layer Normalization).
